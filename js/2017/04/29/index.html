百度 echarts 基本使用

做数据分析，少不了用到 canvas 绘图，echarts 和 chart 和D3 在绘图上的有非常好的表现。百度出品的 echarts，文档清晰，demo 可实时编辑预览，支持个性定制，webpack 按需引入。对首次接触或者英文不太好的开发者非常友好。如果你的业务要兼容 IE6，那么 ehcarts2 可能是你的不二选择。

以下记录自己入门 echarts 的一些经验

引入方式

script 标签引入。除非你要使用所有的 echarts 功能，否则建议根据自己的需求定制，或使用 加速，因为即使是 echarts.min.js 的体积也还是太大了。

使用 webpack 按需引入模块，使打包的体积可控。支持的模块见列表

部分配置参数解释

以一个 dom 节点作为容器，绘制 echarts，用框架开发时，要在 dom 挂载后的钩子中初始化 echarts 实例

更多详细配置，见官方配置项手册，手册的目录结构和配置项一一对应，非常便于查阅！
